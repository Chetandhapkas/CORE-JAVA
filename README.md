# ☕ Java Chapter 1 - Basics

## 📖 Overview
This chapter covers the basic concepts of Java programming. It includes writing the first Java program, performing simple arithmetic operations, and calculating the area of a circle.

## 📂 Programs Included

### 1. Hello Java Program
- Prints a simple message on the screen.
- Introduces the basic structure of a Java program.
- Demonstrates the use of the `main()` method and `System.out.println()`.

### 2. Sum Program
- Accepts two numbers as input.
- Calculates and displays their sum.
- Demonstrates the use of variables, input, and arithmetic operators.

### 3. Area of Circle Program
- Accepts the radius of a circle.
- Calculates the area using the formula:
  **Area = π × r × r**
- Demonstrates mathematical calculations in Java.

## 🎯 Learning Outcomes
After completing this chapter, I learned:
- Java program structure
- `main()` method
- Output using `System.out.println()`
- Variables and data types
- Taking user input
- Arithmetic operators
- Basic mathematical calculations
